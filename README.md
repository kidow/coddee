## Coddee (커디)

> Coddee is a chatting room platform for developers. It provides code block editor with monaco editor.

Link: [https://coddee.dev](https://coddee.dev)

### Project Stack

- Next.js
- Typescript
- TailwindCSS
- Supabase
- Monaco Editor
- Quill
- dayjs
- classnames
- recoil
- heroicons
- cheerio
